BlinkClip quick start

Use BlinkClip when a folder of videos needs the same cleanup: trim dead starts, crop for vertical upload, add an intro or outro, convert files, merge clips, then export the batch from your Mac.

1. Install

  1. Download the current Mac build.
  2. Open the DMG.
  3. Drag BlinkClip into Applications.

2. Run a batch

  1. Select the clips or folder you want to process.
  2. Choose the repeated action.
  3. Export consistent files.

3. Keep originals

During early access, test important jobs with small batches first and keep original files until you verify the exports.

BlinkClip requires macOS 12.0 or newer. No separate FFmpeg install is required for the public Mac app flow.
