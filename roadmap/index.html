Roadmap and feature requests

BlinkClip is early-access Mac software. The roadmap stays focused on repeated video jobs creators already do by hand: batch cleanup, consistent exports, and faster folder-level finishing.

Available now

  • Batch convert, trim, merge, and intro/outro tools.
  • Smart Trim, Serial Crop, and Sequence options.
  • macOS DMG install with in-app updates.

Recently improved

  • Clearer processing feedback after a batch finishes.
  • More reliable release updates and completion states.
  • Better guidance for common creator jobs.

Later

  • More saved sequences for repeated channel work.
  • More controls for social crop passes.
  • Pro features only after the product flow is ready.

Useful requests include the clips you start with, the action you repeat, the output format you need, and where the current process slows down.
